# Rencana Pengembangan Website Toko Online Vape

## Deskripsi Proyek
Website toko online sederhana untuk menjual produk Vape, menggunakan penyimpanan data lokal (IndexedDB).

## Teknologi
- **HTML5**: Struktur halaman.
- **Tailwind CSS**: Desain dan styling (Perlu konfirmasi versi).
- **JavaScript (Vanilla)**: Logika aplikasi.
- **IndexedDB**: Penyimpanan data produk dan keranjang belanja di browser user.

## Fitur Utama
1.  **Halaman Utama (Katalog)**: Menampilkan daftar produk vape (Mod, Pod, Liquid).
2.  **Detail Produk**: Modal atau halaman terpisah untuk detail.
3.  **Keranjang Belanja**: Menambah, mengubah jumlah, dan menghapus item.
4.  **Checkout Sederhana**: Simulasi checkout (karena tidak ada backend).
5.  **Manajemen Data**: Script untuk inisialisasi data produk ke IndexedDB saat pertama kali load.

## Struktur File (Rencana)
- `index.html`: Halaman utama.
- `style.css`: Custom CSS (jika perlu) atau konfigurasi Tailwind.
- `app.js`: Logika utama dan koneksi IndexedDB.
- `db.js`: Module khusus untuk menangani operasi IndexedDB.

## Desain
- **Tema**: Dark Mode dengan aksen Neon (Hijau/Ungu/Biru) untuk nuansa "Vape Store".
- **Layout**: Grid responsif untuk produk.
